=head1 INSTALLATION

Test::JSON::More installation is straightforward. If your CPAN shell is set up,
you should just be able to do

    % cpan Test::JSON::More

Download it, unpack it, then build it as per the usual:

    % perl Makefile.PL
    % make && make test

Then install it:

    % make install
